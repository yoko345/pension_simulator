# 健康保険ロジック設計書（区ベース・完全再現）

---

# 1. 概要

本設計は、年金受給者の手取り計算において必要となる
**健康保険料（国民健康保険・後期高齢者医療制度）を実務レベルで再現するためのロジック定義**である。

対象は以下の2制度：

* 国民健康保険（〜74歳）
* 後期高齢者医療制度（75歳以上）

---

# 2. 前提

## 2.1 保険種別判定

```ts
if (年齢 >= 75) {
  // 後期高齢者医療制度
} else {
  // 国民健康保険
}
```

---

## 2.2 所得の定義

```ts
所得 = max(0, 年金収入 - 公的年金控除 - 430000)
```

---

## 2.3 入力パラメータ

| 項目     | 説明     |
| ------ | ------ |
| 年金収入   | 年額     |
| 年齢     | 月単位で管理 |
| 世帯人数   | 均等割に使用 |
| 介護対象人数 | 40〜64歳 |

---

# 3. 国民健康保険（区）

---

## 3.1 構成

国保は以下の3要素で構成される：

* 基礎分
* 支援金分
* 介護分（40〜64歳のみ）

---

## 3.2 パラメータ

### 基礎分

| 項目  | 値        |
| --- | -------- |
| 所得割 | 7.71%    |
| 均等割 | 47,300円  |
| 上限  | 660,000円 |

---

### 支援金分

| 項目  | 値        |
| --- | -------- |
| 所得割 | 2.69%    |
| 均等割 | 16,800円  |
| 上限  | 260,000円 |

---

### 介護分（40〜64歳）

| 項目  | 値        |
| --- | -------- |
| 所得割 | 2.23%    |
| 均等割 | 16,600円  |
| 上限  | 170,000円 |

---

## 3.3 計算式

### 基礎分

```ts
base =
  min(
    所得 * 0.0771 + 47300 * 世帯人数,
    660000
  )
```

---

### 支援金分

```ts
support =
  min(
    所得 * 0.0269 + 16800 * 世帯人数,
    260000
  )
```

---

### 介護分

```ts
care =
  if (40 <= 年齢 < 65)
    min(
      所得 * 0.0223 + 16600 * 介護対象人数,
      170000
    )
  else
    0
```

---

### 合計

```ts
国保保険料 = base + support + care
```

---

## 4.1 パラメータ

| --- | -------- |

| 均等割 | 47,300円  |
| 上限  | 800,000円 |

---

## 4.2 計算式

```ts
後期医療 =
  min(
    所得 * 0.0967 + 47300,
    800000
  )
```

# 5. 年齢分岐ロジック

```ts
function calcHealthInsurance(params) {

    return calcLateElderlyInsurance(params);
```

# 6. 月次処理への対応

本シミュレーターは月単位で動作するため：

* 年額保険料を算出
* 12で割って月額に変換

```ts
月額保険料 = 年額保険料 / 12
```

---

# 7. 実装上の注意点（重要）

---

## 7.1 上限適用

* 各区分ごとに上限適用すること
* 合算後ではない

---

## 7.2 年齢条件

| 年齢     | 条件         |
| ------ | ---------- |
| 40〜64歳 | 介護保険あり     |
| 65歳以上  | 介護分なし（国保内） |
| 75歳以上  | 後期高齢者      |

---

## 7.3 世帯人数

* 均等割に影響
* UI入力必須

---

## 7.4 軽減制度（未対応）

* 7割・5割・2割軽減あり
* 今回は未実装
