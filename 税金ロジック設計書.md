# 税金ロジック設計書（所得税・住民税 / 実装用最終版）

---

# 1. 概要

本設計は、年金受給者の手取り計算における
**所得税および住民税を実務レベルで再現するためのロジック定義**である。

---

# 2. 前提

---

## 2.1 所得の定義

```ts
所得 = max(0, 年金収入 - 公的年金控除)
```

---

## 2.2 課税所得

```ts
課税所得 = max(0, 所得 - 所得控除合計)
```

---

## 2.3 対象控除

* 基礎控除
* 公的年金等控除
* 配偶者控除 / 配偶者特別控除
* 社会保険料控除
* 生命保険料控除
* 医療費控除

---

# 3. 所得税

---

## 3.1 税率（累進課税）

| 課税所得         | 税率  | 控除額        |
| ------------ | --- | ---------- |
| ～1,950,000円  | 5%  | 0円         |
| ～3,300,000円  | 10% | 97,500円    |
| ～6,950,000円  | 20% | 427,500円   |
| ～9,000,000円  | 23% | 636,000円   |
| ～18,000,000円 | 33% | 1,536,000円 |
| ～40,000,000円 | 40% | 2,796,000円 |
| 40,000,000円超 | 45% | 4,796,000円 |

---

## 3.2 計算式

```ts
所得税 = 課税所得 × 税率 − 控除額
```

---

## 3.3 実装

```ts
function calcIncomeTax(taxableIncome) {
  if (taxableIncome <= 1950000) return taxableIncome * 0.05;

  if (taxableIncome <= 3300000)
    return taxableIncome * 0.1 - 97500;

  if (taxableIncome <= 6950000)
    return taxableIncome * 0.2 - 427500;

  if (taxableIncome <= 9000000)
    return taxableIncome * 0.23 - 636000;

  if (taxableIncome <= 18000000)
    return taxableIncome * 0.33 - 1536000;

  if (taxableIncome <= 40000000)
    return taxableIncome * 0.4 - 2796000;

  return taxableIncome * 0.45 - 4796000;
}
```

---

## 3.4 復興特別所得税

```ts
所得税 *= 1.021
```

# 4. 住民税

---

## 4.1 基本構造

* 所得割：10%
* 均等割：5,000円（自治体により若干変動）

---

## 4.2 計算式

```ts
住民税 = 所得割 + 均等割
```

---

## 4.3 所得割

```ts
所得割 = 課税所得 × 0.10
```

---

## 4.4 均等割

```ts
均等割 = 5000
```

---

## 4.5 実装

```ts
function calcResidentTax(taxableIncome) {
  const incomePortion = taxableIncome * 0.10;
  const flatPortion = 5000;

  return incomePortion + flatPortion;
}
```

---

# 5. 非課税判定（重要）

---

## 5.1 判定条件（簡易）

```ts
if (所得 <= 非課税ライン) {
  住民税 = 0;
}
```

---

## 5.2 非課税ライン（簡易モデル）

```ts
非課税ライン = 450000 + (350000 × 世帯人数)
```

---

## 5.3 実装

```ts
function isResidentTaxExempt(income, householdSize) {
  const threshold = 450000 + 350000 * householdSize;
  return income <= threshold;
}
```

# 8. 注意書き（必須表示）

---

## ■ 年金収入のみを前提とした計算について

本シミュレーションでは、**年金収入のみ**を前提として税額を算出しています。

実際には以下の要素により税額が変動します：

* 給与収入
* 不動産収入
* 配当所得
* 配偶者の所得
* 各種控除の詳細条件

これらがある場合、本結果と実際の税額は異なる可能性があります。

---

## ■ 控除の簡略化について

本ツールでは以下の控除を簡略化して扱っています：

* 控除上限の詳細条件
* 適用判定の細かい分岐

そのため、実際の税額と差異が生じる可能性があります。

---

## ■ 非課税判定について

住民税の非課税判定は簡易モデルを使用しています。

実際には：

* 扶養状況
* 障害者控除
* 寡婦控除等

により判定が変動します。

---

## ■ 制度変更について

本モデルは2026年時点の制度を前提としています。

税制改正により、将来的に税率・控除は変更される可能性があります。
